# GroundControl

GenAI stadium operations & fan copilot — an unofficial concept build for a FIFA World
Cup 2026 stadium-operations hackathon submission. Not affiliated with or endorsed by
FIFA.

## What it does
A multilingual fan assistant for wayfinding, accessibility routing, and green transport
choices, paired with a staff/volunteer ops dashboard that turns live crowd signals into
ready-to-approve actions — reroutes, alerts, incident responses, shift briefings.
